IP-SAN Quiz

1. What is the primary function of IP SAN?

غلط: الـ IP SAN مش بينقل الـ file-level data، ده بينقل الـ block-level data. الإجابة الصحيحة هي c.
غلط: الـ IP SAN مش مخصص لربط الخدمات السحابية بالـ virtual machines. الإجابة الصحيحة هي c.
صح: الـ IP SAN بيستخدم بروتوكولات زي الـ iSCSI وFCIP لنقل الـ block-level data عبر شبكات الـ IP.
غلط: الـ IP SAN مش بيدعم الـ object storage ولا بيشتغل عبر FC. الإجابة الصحيحة هي c.

2. Which protocol is commonly used in IP SAN for transporting SCSI data?

غلط: الـ FTP بيستخدم لنقل الـ files، مش الـ SCSI data. الإجابة الصحيحة هي b.
صح: الـ iSCSI هو البروتوكول الأساسي في الـ IP SAN لنقل الـ SCSI data عبر الـ TCP/IP.
غلط: الـ NFS بيستخدم لنقل الـ file-level data، مش الـ SCSI. الإجابة الصحيحة هي b.
غلط: الـ SMB بيستخدم لمشاركة الملفات، مش لنقل الـ SCSI data. الإجابة الصحيحة هي b.

3. What is a key advantage of IP SAN over FC SAN?

صح: الـ IP SAN بيستخدم الشبكات الجاهزة بتاعت الـ IP، فبيكون أرخص من الـ FC SAN اللي محتاج معدات مخصصة.
غلط: الـ FC SAN بيوفر أداء أعلى للـ I/O operations. الإجابة الصحيحة هي a.
غلط: الـ IP SAN مش محتاج هاردوير مخصص زي الـ FC SAN. الإجابة الصحيحة هي a.
غلط: الـ IP SAN بيدعم مسافات طويلة. الإجابة الصحيحة هي a.

4. Which component in iSCSI is responsible for initiating data transfer?

غلط: الـ iSCSI target هو جهاز التخزين اللي بيستقبل البيانات. الإجابة الصحيحة هي c.
غلط: الـ iSCSI port مش مكون يبدأ النقل. الإجابة الصحيحة هي c.
صح: الـ iSCSI initiator هو اللي بيبدأ نقل البيانات بإرسال الأوامر للـ iSCSI target.
غلط: الـ iSNS server بيساعد في اكتشاف الأجهزة، مش بيبدأ النقل. الإجابة الصحيحة هي c.

5. Which type of iSCSI initiator is least expensive to implement?

غلط: الـ TOE NIC بيحتاج هاردوير مخصص، فهو أغلى. الإجابة الصحيحة هي b.
صح: الـ Software iSCSI adapter هو الأرخص لأنه بيعتمد على سوفتوير وكارت شبكة عادي.
غلط: الـ iSCSI HBA بيحتاج هاردوير مخصص وأغلى. الإجابة الصحيحة هي b.
غلط: الـ FCIP gateway مش initiator أصلاً. الإجابة الصحيحة هي b.

6. Which iSCSI initiator type offloads both iSCSI and TCP/IP processing?

صح: الـ iSCSI HBA بيفرّغ معالجة الـ iSCSI والـ TCP/IP من الـ CPU.
غلط: الـ Software iSCSI adapter بيعتمد على الـ CPU للمعالجة. الإجابة الصحيحة هي a.
غلط: الـ TOE NIC بيفرّغ بس الـ TCP/IP، مش الـ iSCSI. الإجابة الصحيحة هي a.
غلط: الـ FC initiator مش بيستخدم في الـ iSCSI. الإجابة الصحيحة هي a.

7. What is a disadvantage of using a software iSCSI adapter?

غلط: الـ Software iSCSI adapter مش محتاج هاردوير مخصص. الإجابة الصحيحة هي c.
غلط: الـ Software iSCSI adapter هو الأرخص. الإجابة الصحيحة هي c.
صح: الـ Software iSCSI adapter بيحط حمل زيادة على الـ CPU لمعالجة الـ iSCSI والـ TCP/IP.
غلط: الـ Software iSCSI adapter مش محدود بالمسافة. الإجابة الصحيحة هي c.

8. Which connectivity type requires a gateway to connect iSCSI to FC SAN?

غلط: الـ Native iSCSI بيتصل مباشرة من غير gateway. الإجابة الصحيحة هي b.
صح: الـ Bridged iSCSI محتاج gateway لتحويل الـ IP packets لـ FC frames.
غلط: الـ Direct FC مش بيستخدم في الـ iSCSI. الإجابة الصحيحة هي b.
غلط: الـ iSNS بيساعد في الاكتشاف، مش بيحتاج gateway. الإجابة الصحيحة هي b.

9. What is the purpose of iSCSI discovery?

صح: الـ iSCSI discovery بيحدد مكان واسم الـ iSCSI targets في الشبكة باستخدام SendTargets أو iSNS.
غلط: الـ iSCSI discovery مش بيشفر البيانات. الإجابة الصحيحة هي a.
غلط: تكوين الـ storage volumes مش من مهام الـ discovery. الإجابة الصحيحة هي a.
غلط: دمج الـ FC fabrics مش من مهام الـ iSCSI. الإجابة الصحيحة هي a.

10. Which discovery method requires manual configuration of the target's network portal?

غلط: الـ iSNS discovery بيشتغل أوتوماتيك. الإجابة الصحيحة هي c.
غلط: الـ FC zoning مش طريقة اكتشاف في الـ iSCSI. الإجابة الصحيحة هي c.
صح: الـ SendTargets discovery بيحتاج تكوين يدوي للـ network portal بتاع الـ target.
غلط: الـ FCIP tunneling مش طريقة اكتشاف. الإجابة الصحيحة هي c.

11. What is the role of the iSNS server in iSCSI?

غلط: الـ iSNS server مش بيشفر البيانات. الإجابة الصحيحة هي b.
صح: الـ iSNS server بيسجّل الـ initiators والـ targets علشان يسهل اكتشافهم.
غلط: تحويل الـ FC frames لـ IP packets بيتم في الـ FCIP. الإجابة الصحيحة هي b.
غلط: إدارة الـ storage volumes مش من مهام الـ iSNS. الإجابة الصحيحة هي b.

12. How are iSNS discovery domains similar to FC zones?

صح: الـ iSNS discovery domains بتجمّع الأجهزة للتواصل زي الـ FC zones في الـ FC SAN.
غلط: لا الـ iSNS domains ولا الـ FC zones بيشفروا البيانات. الإجابة الصحيحة هي a.
غلط: إدارة السعة مش من مهام الـ iSNS domains أو FC zones. الإجابة الصحيحة هي a.
غلط: تفريغ الـ CPU مش من مهام الـ iSNS أو FC zones. الإجابة الصحيحة هي a.

13. What does the iSNS server use to notify devices of network events?

غلط: الـ FCIP tunnels بيستخدم لربط الـ FC SANs، مش للإشعارات. الإجابة الصحيحة هي c.
غلط: الـ SendTargets بيستخدم للاكتشاف، مش الإشعارات. الإجابة الصحيحة هي c.
صح: الـ iSNS server بيستخدم State Change Notifications (SCNs) لإشعار الأجهزة بالتغييرات في الشبكة.
غلط: الـ VE_Ports بتستخدم في الـ FCIP، مش الـ iSNS. الإجابة الصحيحة هي c.

14. What is the primary function of FCIP?

غلط: الـ FCIP مش بينقل الـ file-level data. الإجابة الصحيحة هي b.
صح: الـ FCIP بيربط شبكات الـ FC SAN عبر الـ IP باستخدام tunnels.
غلط: إدارة الـ iSCSI initiators مش من مهام الـ FCIP. الإجابة الصحيحة هي b.
غلط: الـ FCIP مش بيشفر الـ SCSI data. الإجابة الصحيحة هي b.

15. Which port type is used in FCIP to connect to an FC switch without merging fabrics?

صح: الـ EX_Port بيستخدم في الـ FCIP لربط سويتش من غير دمج الـ fabrics.
غلط: الـ VE_Port بيدمج الـ fabrics. الإجابة الصحيحة هي a.
غلط: الـ iSCSI Port مش بيستخدم في الـ FCIP. الإجابة الصحيحة هي a.
غلط: الـ E_Port بيستخدم في الـ FC العادي، مش الـ FCIP. الإجابة الصحيحة هي a.

16. What is the role of VE_Port in FCIP?

غلط: الـ VE_Port مش بيشفر الـ FC frames. الإجابة الصحيحة هي b.
صح: الـ VE_Port بيعمل virtual ISLs لدمج الـ fabrics في الـ FCIP.
غلط: إدارة الـ iSCSI targets مش من مهام الـ VE_Port. الإجابة الصحيحة هي b.
غلط: تفريغ الـ TCP/IP بيتم بواسطة هاردوير زي TOE NIC. الإجابة الصحيحة هي b.

17. What is a key benefit of IP SAN for disaster recovery?

غلط: الـ FC SAN بيوفر أداء أعلى للـ I/O. الإجابة الصحيحة هي c.
غلط: الـ IP SAN مش محتاج هاردوير مخصص. الإجابة الصحيحة هي c.
صح: الـ IP SAN بيمد الشبكات لمسافات طويلة عبر الـ IP networks، وده مثالي للـ disaster recovery.
غلط: الـ IP SAN مش بيبسّط إدارة التخزين بالضرورة. الإجابة الصحيحة هي c.

18. Which component converts IP packets to FC frames in Bridged iSCSI?

صح: الـ gateway أو multiprotocol router بيحول الـ IP packets لـ FC frames في الـ Bridged iSCSI.
غلط: الـ iSCSI initiator بيبدأ النقل، مش بيحول البيانات. الإجابة الصحيحة هي a.
غلط: الـ iSNS server بيساعد في الاكتشاف، مش التحويل. الإجابة الصحيحة هي a.
غلط: الـ VE_Port بيستخدم في الـ FCIP، مش الـ Bridged iSCSI. الإجابة الصحيحة هي a.

19. What does an iSCSI address consist of?

غلط: الـ FC port وLUN بيستخدموا في الـ FC SAN. الإجابة الصحيحة هي b.
صح: عنوان الـ iSCSI بيتكون من الـ IP address وiSCSI name مع الـ TCP port.
غلط: الـ VE_Port بيستخدم في الـ FCIP، مش الـ iSCSI. الإجابة الصحيحة هي b.
غلط: الـ FC frame وiSNS server مش جزء من عنوان الـ iSCSI. الإجابة الصحيحة هي b.

20. Which protocol encapsulates FC frames over TCP/IP in FCIP?

غلط: الـ iSCSI بينقل الـ SCSI، مش الـ FC frames. الإجابة الصحيحة هي c.
غلط: الـ iSNS بيساعد في الاكتشاف، مش التغليف. الإجابة الصحيحة هي c.
صح: الـ FCIP بيغلّف الـ FC frames عبر الـ TCP/IP لربط الـ FC SANs.
غلط: الـ SCSI هو بروتوكول علوي، مش بيغلّف الـ FC frames. الإجابة الصحيحة هي c.

21. IP SAN leverages existing IP-based network infrastructure.

صح: الـ IP SAN بيستخدم الشبكات الجاهزة بتاعت الـ IP، فبيكون أوفر من الـ FC SAN.
غلط: الإجابة الصحيحة هي True لأن الـ IP SAN بيستخدم البنية التحتية للـ IP.

22. iSCSI transports file-level data over IP networks.

غلط: الـ iSCSI بينقل الـ block-level data، مش الـ file-level data. الإجابة الصحيحة هي False.
صح: الـ iSCSI بينقل الـ block-level data عبر الـ IP networks.

23. iSCSI HBA offloads both iSCSI and TCP/IP processing from the CPU.

صح: الـ iSCSI HBA بيفرّغ معالجة الـ iSCSI والـ TCP/IP من الـ CPU.
غلط: الإجابة الصحيحة هي True لأن الـ iSCSI HBA بيفرّغ المعالجة.

24. Native iSCSI requires a gateway to connect to storage systems.

غلط: الـ Native iSCSI بيتصل مباشرة بالـ storage systems من غير gateway. الإجابة الصحيحة هي False.
صح: الـ Native iSCSI مش محتاج gateway، على عكس الـ Bridged iSCSI.

25. SendTargets discovery requires manual configuration of the target's network portal.

صح: الـ SendTargets discovery بيحتاج تكوين يدوي للـ network portal بتاع الـ target.
غلط: الإجابة الصحيحة هي True لأن الـ SendTargets بيحتاج تكوين يدوي.

26. iSNS discovery domains group devices for communication in IP SAN.

صح: الـ iSNS discovery domains بتجمّع الأجهزة في الـ IP SAN للتواصل، زي الـ FC zones.
غلط: الإجابة الصحيحة هي True لأن الـ iSNS domains بتجمّع الأجهزة.

27. FCIP is used to transport SCSI data directly over IP networks.

غلط: الـ FCIP بينقل الـ FC frames، مش الـ SCSI data مباشرة. الإجابة الصحيحة هي False.
صح: الـ FCIP بيغلّف الـ FC frames، والـ iSCSI هو اللي بينقل الـ SCSI data.

28. FCIP tunnels can merge FC fabrics into a single large fabric.

صح: الـ FCIP tunnels بتدمج الـ FC fabrics في شبكة واحدة كبيرة باستخدام VE_Ports.
غلط: الإجابة الصحيحة هي True لأن الـ FCIP بيدعم دمج الـ fabrics.

29. EX_Port in FCIP enables routing without merging fabrics.

صح: الـ EX_Port في الـ FCIP بيسمح بالـ routing من غير دمج الـ fabrics.
غلط: الإجابة الصحيحة هي True لأن الـ EX_Port بيدعم الـ routing.

30. iSCSI HBA requires additional CPU cycles for iSCSI processing.

غلط: الـ iSCSI HBA بيفرّغ معالجة الـ iSCSI من الـ CPU. الإجابة الصحيحة هي False.
صح: الـ iSCSI HBA مش بيحتاج دورات إضافية من الـ CPU لمعالجة الـ iSCSI.

31. IP SAN supports long-distance disaster recovery solutions.

صح: الـ IP SAN بيدعم حلول الـ disaster recovery لمسافات طويلة عبر الـ IP networks.
غلط: الإجابة الصحيحة هي True لأن الـ IP SAN بيمد الشبكات لمسافات طويلة.

32. TOE NIC handles iSCSI processing without CPU involvement.

غلط: الـ TOE NIC بيفرّغ بس الـ TCP/IP، لكن الـ iSCSI processing بتحتاج الـ CPU. الإجابة الصحيحة هي False.
صح: الـ TOE NIC مش بيفرّغ الـ iSCSI processing، فبيحتاج الـ CPU.

33. iSCSI targets can be storage systems with iSCSI ports.

صح: الـ iSCSI targets بتكون أنظمة تخزين زي الـ storage systems فيها iSCSI ports.
غلط: الإجابة الصحيحة هي True لأن الـ iSCSI targets فعلاً بتكون أنظمة تخزين زي الـ storage systems اللي فيها iSCSI ports.
